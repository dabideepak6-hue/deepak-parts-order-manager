# Deepak Parts Order Manager

A professional Spare Parts Order Management System built with HTML, CSS, and JavaScript.

## Features

- DMS Order Number
- Customer Name
- Mobile Number
- Part Number
- Part Name
- Quantity
- Order Date
- Expected Delivery Date
- Order Status (Pending, Ordered, Delivered, Cancelled)
- Search & Filter
- Edit & Delete Orders
- Local Storage (No Firebase)
- WhatsApp Status Update
- CSV Export
- Print Support
- Mobile Responsive
- Offline Support

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
